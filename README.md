# m.n.k-games-solver
## The program supports 3 commands:
###  GEN_ALL_POS_MOV N M K ActivePlayer -  generating all possible moves along with their number
## Example:
### Input
```
GEN_ALL_POS_MOV 3 3 3 2
1 0 0
0 0 0
0 0 0
```
